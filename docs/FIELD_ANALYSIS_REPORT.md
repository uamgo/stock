#### 2. 概念热度计算方法重新设计

✅ **新版4维度热度计算**：
1. **板块涨跌幅 (40%)** - 反映概念整体表现
2. **主力资金净流入 (30%)** - 反映资金关注度  
3. **成交活跃度 (20%)** - 反映市场参与度
4. **技术指标 (10%)** - 反映技术面强度

✅ **计算公式优化**：
- 板块涨跌幅得分 = clip((涨跌幅+10)*5, 0, 100)
- 资金流入得分 = clip((净流入/亿)*2+50, 0, 100)
- 成交活跃度得分 = clip(log10(成交额/亿+1)*20, 0, 100)
- 技术指标得分 = clip(abs(振幅)/百万*5, 0, 100)
